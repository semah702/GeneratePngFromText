# write-text-with-python
Generate PNG file from Text.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install Pillow
```

## Usage

```python
text = "Hello, world!" # You can change text here.
```
After run ```text2Png.py``` file.
